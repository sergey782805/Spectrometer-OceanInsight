# QtWidgetsApplication1
Программа для работы со спектрометром от компании Ocean Optic с возможностью перерасчета спектра в расширенном диапазоне ФАР (350 - 800 нм) в PPFD (Плотность потока фотосентетических потоков).

В программе использовались:
1. QT framework 6.10, проект написан на QT widget
2. SDK Ocean Direct на C++.
